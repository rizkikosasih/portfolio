### 4. Description (Deskripsi Proyek)
*Salin teks di bawah ini (Pilihlah salah satu bahasa yang Anda gunakan di LinkedIn):*

#### 🇮🇩 Versi Bahasa Indonesia:
```text
Portfolio v2 adalah showcase teknis interaktif yang dirancang untuk menggambarkan transisi karir saya dari PHP Native menuju ekosistem Web Modern Full-Stack/Backend. Fokus utama proyek ini adalah Type Safety, Data Integrity, dan Scalable Architecture.

Fitur & Keunggulan Utama:
- Interactive & Responsive UI: Dilengkapi galeri proyek horizontal yang mendukung drag-to-scroll (seret mouse) dengan kustomisasi snap alignment, serta menu navigasi HP yang responsif menggunakan Framer Motion.
- Technical Excellence (Backend Mindset): Integrasi penuh Sanity Typegen untuk memastikan Zero-Any policy pada data CMS, serta proteksi Data Integrity (Nullish Coalescing) untuk mencegah crash.
- Offline Caching & Mock System: Implementasi data mock lokal untuk efisiensi pengembangan offline tanpa memakan kuota API Sanity CMS.
- Performance First: Kecepatan memuat yang optimal dengan optimalisasi gambar on-the-fly (@sanity/image-url) dan pembersihan modul yang tidak terpakai (tree-shaking).

Stack Teknologi: React 19, Vite, TypeScript, Tailwind CSS, Shadcn UI, Framer Motion, Sanity.io (Headless CMS).
```

#### 🇬🇧 Versi Bahasa Inggris (Rekomendasi untuk jangkauan HR global):
```text
Portfolio v2 is a modern personal showcase designed to highlight my professional transition from PHP Native development to a modern Full-Stack/Backend ecosystem. The architecture is heavily engineered around Type Safety, Data Integrity, and Performance.

Key Features & Implementations:
- Fluid Draggable Gallery: Custom drag-to-scroll desktop behavior on the Projects gallery with snap-alignment and drag-start overrides to prevent accidental click triggers.
- Mobile Navigation Menu: Animated, glassmorphic dropdown panel utilizing React 19 and Framer Motion's AnimatePresence for optimal mobile responsiveness.
- Zero-Any Policy & Type Safety: Full integration with Sanity Typegen for automated type safety across all CMS schemas.
- High-Performance Caching: Offline development mocks to bypass live API calls, combined with modern TSConfig structures.

Tech Stack: React 19, Vite, TypeScript, Tailwind CSS, Shadcn UI, Framer Motion, Sanity.io (Headless CMS).
```
