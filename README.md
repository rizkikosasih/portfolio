# Portfolio Website v1 - 3D Interactive Design

## 📝 Overview

Dokumentasi ini adalah untuk versi pertama (v1) dari website portofolio saya. Versi ini berfokus pada **Wow Effect** dengan implementasi objek 3D interaktif menggunakan **Three.js**. Website ini dirancang untuk memperkenalkan profil profesional saya sebagai pengembang web yang memiliki pemahaman kuat pada fundamental pemrograman.

---

## 🛠️ Tech Stack

Versi ini dibangun dengan teknologi berikut:

- **Core:** React.js

- **3D Engine:** Three.js, @react-three/fiber, @react-three/drei

- **Styling:** Tailwind CSS

- **Animations:** Framer Motion

- **Contact:** @emailjs/browser

---

## 🏗️ Folder Structure

Struktur proyek ini bersifat modular dengan pembagian halaman (Multi-page):

- `Home`: Menampilkan 3D object scene (Tree) sebagai elemen visual utama.

- `Tentang`: Berisi narasi perkenalan diri dan riwayat pendidikan.

- `Kemampuan`: Daftar ikon teknologi yang dikuasai secara individual.

- `Hasil Karya`: Showcase proyek-proyek utama seperti CRM WhatsApp Web dan Task Management API.

- `Pengalaman Kerja`: Menampilkan ringkasan pengalaman profesional dalam pengembangan aplikasi backend dan sistem internal.

- `Kontak`: Form komunikasi yang terintegrasi dengan EmailJS.

---

## 🚀 Key Features

1. **Interactive 3D Scene:** Visualisasi objek 3D di halaman utama menggunakan React Three Fiber.

2. **Modular Components:** Pemisahan komponen UI untuk navigasi, footer, dan konten utama.

3. **Responsive Design:** Tampilan yang dioptimalkan untuk berbagai perangkat.
